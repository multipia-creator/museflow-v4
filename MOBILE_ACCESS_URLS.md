## 🛠️ DNS 캐시 문제 해결

### Android
1. **Chrome 캐시 삭제**:
   - 설정 → 개인정보 보호 → 인터넷 사용 기록 삭제
   - "캐시된 이미지 및 파일" 선택

2. **DNS 리셋**:
   - 설정 → 네트워크 및 인터넷 → Wi-Fi
   - 연결된 네트워크 탭 → "고급"
   - Private DNS: 자동 → `dns.google` → 다시 자동

### iOS
1. **Safari 캐시 삭제**:
   - 설정 → Safari
   - "방문 기록 및 웹사이트 데이터 지우기"

2. **DNS 설정**:
   - 설정 → Wi-Fi → (i) 버튼
   - DNS 구성 → 수동
   - `8.8.8.8`, `8.8.4.4` 추가
