## 🎮 샘플 화면 사용법

### **1. AI Chat 접기/펼치기**
```
1. 좌측 사이드바에서 "AI Conversation" 클릭
2. 접으면 → 48px만 차지 (공간 절약)
3. 펼치면 → 300px 차트 영역
```

### **2. 캔버스 아이템 이동**
```
1. 이미지/텍스트/차트 위에 마우스 올리기
2. 드래그하여 원하는 위치로 이동
3. 자동으로 20px 그리드에 스냅됨
4. 놓으면 위치 고정
```

### **3. 아이템 조작**
```
각 아이템 우측 상단:
- [📋] 클릭 → 복사
- [🔄] 클릭 → 이동 모드
- [🗑️] 클릭 → 삭제
```

### **4. 줌 컨트롤**
```
우측 하단 패널:
- [−] 클릭 → 축소
- [+] 클릭 → 확대
- [🔄] 클릭 → 100%로 리셋
```
