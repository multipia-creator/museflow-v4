## ✨ Canvas 주요 기능

### 1️⃣ **3가지 뷰 모드**
- **Timeline View**: 단계별 수평 레이아웃
- **Kanban Board**: 드래그 앤 드롭 가능한 칸반 보드
- **Gallery View**: 프로젝트 갤러리 뷰

### 2️⃣ **Drag & Drop**
- 작업 카드를 다른 단계로 드래그하여 이동
- 실시간 API 호출로 D1 Database 업데이트
- 부드러운 애니메이션 효과

### 3️⃣ **Checklist 관리**
- 작업별 체크리스트 추가/삭제
- 실시간 진행률 계산 및 표시
- 진행률 막대 시각화

### 4️⃣ **담당자 & 마감일**
- 각 작업에 담당자 배정
- 마감일 설정 및 표시
- D-Day 알림 (향후 기능)

### 5️⃣ **실시간 저장**
- 모든 변경사항 즉시 D1 Database 저장
- localStorage 의존성 제거
- 브라우저/디바이스 간 동기화
