## 소개

MuseFlow AI Orchestrator는 큐레이터의 업무를 AI가 자동화하는 혁신적인 시스템입니다.

### 핵심 가치
- ⏱️ **87.5% 시간 절감**: 8시간 업무를 1시간으로 단축
- 🤖 **95% AI 자동화**: AI가 대부분의 작업을 자동으로 처리
- 🎯 **큐레이터는 최종 결정만**: 전략적 의사결정에 집중

### 지원 업무
1. **전시 기획** (1시간 자율 실행)
2. **예산 승인** (94% 시간 절감)
3. **소장품 선정** (하이브리드 AI)
4. **교육 프로그램** (자동 커리큘럼 생성)
5. **홍보 계획** (SNS, 보도자료 자동 생성)

## 빠른 시작

### 1단계: 대시보드 접속
[AI Orchestrator Dashboard](http://localhost:3000/ai-orchestrator-dashboard.html)에 접속합니다.

### 2단계: 워크플로우 선택
5가지 워크플로우 중 하나를 선택합니다:
- 🎨 **전시 기획**
- 💰 **예산 승인**
- 🖼️ **소장품 선정**
- 🎓 **교육 프로그램**
- 📢 **홍보 계획**

### 3단계: AI에게 명령하기
자연어로 명령을 입력합니다:
```
"인상주의 특별전을 3000만원 예산으로 기획해줘. 
모네, 르누아르, 마네 작품 중심으로 구성하고, 
3개월 기간으로 계획해줘."
```

### 4단계: 실시간 진행 확인
- Phase별 진행 상황 실시간 표시
- Canvas 노드 자동 생성
- Dashboard 위젯 자동 업데이트

### 5단계: 결과 확인 및 승인
- 생성된 문서 다운로드
- Canvas에서 노드 확인
- Dashboard에서 위젯 확인

## 주요 기능

### 🤖 AI Multi-Agent 시스템
6개의 전문 AI Agent가 협업합니다:

#### 1. Research Agent
- **역할**: 작품, 전시, 트렌드 조사
- **기능**: Wikipedia, MET Museum, Gemini AI 검색
- **출력**: 조사 보고서, AI 요약

#### 2. Canvas Agent
- **역할**: Canvas 노드 자동 생성
- **기능**: 전시 컨셉, 예산 차트, 작품 노드 생성
- **출력**: 시각화된 Canvas 노드

#### 3. Document Agent
- **역할**: 문서 자동 작성
- **기능**: 예산서, 홍보 계획, SNS 콘텐츠 생성
- **출력**: Markdown/PDF 문서

#### 4. Widget Agent
- **역할**: Dashboard 위젯 생성
- **기능**: 예산 차트, 작품 갤러리, 일정 위젯
- **출력**: 인터랙티브 위젯

#### 5. Integration Agent
- **역할**: 외부 API 연동
- **기능**: Wikipedia, MET Museum, Google Search
- **출력**: 통합 데이터

#### 6. Monitor Agent
- **역할**: 실시간 모니터링
- **기능**: 성능 추적, 에러 감지, 알림
- **출력**: 메트릭, 알림

## 3가지 핵심 시나리오

### Scenario A: 예산 승인 (94% 시간 절감)

**Before**: 90분 소요
- 과거 예산 데이터 검색 (20분)
- 예산 계획서 작성 (40분)
- 승인 이메일 작성 및 전송 (20분)
- 회의 일정 조율 (10분)

**After**: 5분 소요
- AI가 모든 과정을 자동으로 처리
- 큐레이터는 최종 승인만 클릭

**실행 방법**:
1. **워크플로우 선택**: "예산 승인"
2. **명령 입력**: "3000만원 예산으로 인상주의 전시 예산 계획서를 작성해줘"
3. **자동 실행**: AI가 다음을 자동으로 처리
   - 과거 예산 데이터 분석
   - 5개 항목별 예산 자동 계산 (작품 대여 50%, 보험 20%, 디자인 15%, 홍보 10%, 기타 5%)
   - 예산 계획서 자동 생성 (Markdown)
   - 예산 비교 차트 위젯 생성
4. **결과 확인**: 
   - 문서: 예산 계획서.md
   - 위젯: Budget Comparison Chart
   - Canvas: Budget Chart 노드

---

### Scenario B: 전시 기획 (1시간 자율 실행)

**Before**: 8시간 소요
- 트렌드 조사 (2시간)
- 작품 선정 (3시간)
- 컨셉 개발 (2시간)
- 기획서 작성 (1시간)

**After**: 1시간 소요
- AI가 7단계 기획 프로세스를 자동으로 실행
- 3가지 전시 옵션을 제시
- 큐레이터는 최종 옵션만 선택

**실행 방법**:
1. **워크플로우 선택**: "전시 기획"
2. **명령 입력**: "인상주의 특별전을 3개월 기간으로 기획하고 3가지 옵션을 제시해줘"
3. **AI 자율 실행 (7 Phase)**:
   - Phase 1: Research (작품 및 트렌드 조사)
   - Phase 2: Concept Generation (3가지 컨셉 생성)
   - Phase 3: Artwork Selection (작품 선정)
   - Phase 4: Budget Calculation (예산 계산)
   - Phase 5: Document Generation (기획서 작성)
   - Phase 6: Promotion Planning (홍보 계획)
   - Phase 7: Final Review (최종 검토)
4. **결과 확인**:
   - 문서: 전시 기획서.md, 홍보 계획서.md
   - 위젯: Exhibition Calendar, Budget Chart
   - Canvas: 3가지 컨셉 노드, 작품 노드

---

### Scenario C: 소장품 선정 (하이브리드 AI)

**Before**: 4시간 소요
- 소장품 DB 검색 (1시간)
- 작품 분석 (2시간)
- 목록 작성 (1시간)

**After**: 30분 소요
- AI가 10개 작품을 자동 선정
- Canvas 노드로 시각화
- 큐레이터가 최종 승인

**실행 방법**:
1. **워크플로우 선택**: "소장품 선정"
2. **명령 입력**: "인상주의 관련 소장품 10개를 선정하고 갤러리 위젯을 만들어줘"
3. **AI 자동 선정**:
   - Internal DB에서 인상주의 작품 검색
   - 관련도 점수 계산
   - Top 10 작품 자동 선정
4. **결과 확인**:
   - Canvas: 10개 작품 노드 (3x2 그리드)
   - 위젯: Artwork Gallery
   - 각 작품 정보: 제목, 작가, 연도, 크기, 대여비

## UI 가이드

### Dashboard AI Command Panel
**URL**: `/ai-orchestrator-dashboard.html`

**주요 기능**:
- 5가지 워크플로우 카드 선택
- AI 명령 입력 (자연어)
- 실시간 진행률 표시
- Phase별 진행 상태
- 실시간 이벤트 로그
- 결과 다운로드

**사용 팁**:
- 워크플로우를 먼저 선택하세요
- 명령은 구체적으로 입력하세요 (예산, 기간, 작품 등)
- 진행 중에는 다른 작업을 하지 마세요

---

### Canvas V4 Hybrid Layout
**URL**: `/canvas-v4-hybrid.html`

**3-Column 레이아웃**:
1. **좌측 (History & Projects)**
   - 최근 세션 히스토리
   - 클릭으로 세션 전환
   
2. **중앙 (AI Conversation)**
   - AI와 대화형 인터페이스
   - 실시간 Phase 진행 표시
   - 메시지 입력
   
3. **우측 (Live Canvas/Dashboard Preview)**
   - Canvas/Dashboard 탭 전환
   - 실시간 노드/위젯 표시

**사용 팁**:
- 대화형 인터페이스로 자연스럽게 명령
- Enter 키로 메시지 전송
- 우측 패널에서 실시간 결과 확인

---

### Real-time Sync Dashboard
**URL**: `/ai-orchestrator-sync.html`

**2-Panel 동기화**:
1. **좌측 (Dashboard Widgets)**
   - 위젯 실시간 표시
   - Sync 상태 배지
   
2. **우측 (Canvas Nodes)**
   - 노드 실시간 표시
   - Dashboard로 동기화 화살표

**사용 팁**:
- "AI 데모 실행" 버튼으로 테스트
- 좌우 패널 동시 확인
- Sync 애니메이션 관찰

## FAQ

### Q1: AI가 생성한 문서를 수정할 수 있나요?
**A**: 네, 생성된 문서는 Markdown 형식이므로 자유롭게 수정 가능합니다. 다운로드 후 편집기에서 수정하세요.

### Q2: AI 실행 중 중단할 수 있나요?
**A**: 네, "Cancel" 버튼을 클릭하면 언제든지 중단할 수 있습니다.

### Q3: 여러 워크플로우를 동시에 실행할 수 있나요?
**A**: 현재는 한 번에 하나의 워크플로우만 실행 가능합니다. 순차적으로 실행해주세요.

### Q4: AI가 만든 결과를 어디서 확인하나요?
**A**: 3곳에서 확인할 수 있습니다:
- **Dashboard**: 위젯으로 표시
- **Canvas**: 노드로 표시
- **문서**: "문서 다운로드" 버튼

### Q5: AI가 실패하면 어떻게 하나요?
**A**: 자동으로 최대 3회 재시도합니다. 그래도 실패하면 에러 메시지를 확인하고 다시 시도하세요.

### Q6: 과거 실행 히스토리를 볼 수 있나요?
**A**: 네, Canvas V4 Hybrid Layout의 좌측 패널에서 최근 히스토리를 볼 수 있습니다.

## 문제 해결

### 문제 1: AI가 응답하지 않습니다
**원인**: 네트워크 연결 문제 또는 서버 과부하

**해결**:
1. 브라우저 콘솔(F12)에서 에러 확인
2. 페이지 새로고침
3. 서버 상태 확인 (`/api/orchestrator/templates`)

---

### 문제 2: SSE 연결이 끊어집니다
**원인**: 방화벽 또는 프록시 설정

**해결**:
1. 브라우저 콘솔에서 SSE 에러 확인
2. 방화벽 설정 확인
3. 다른 브라우저로 시도

---

### 문제 3: 결과가 표시되지 않습니다
**원인**: 캐시 문제 또는 JavaScript 에러

**해결**:
1. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
2. 하드 리프레시 (Ctrl+F5)
3. 브라우저 콘솔에서 JavaScript 에러 확인

---

### 문제 4: 생성된 문서가 이상합니다
**원인**: 입력 명령이 불명확하거나 AI 학습 데이터 부족

**해결**:
1. 더 구체적인 명령 입력 (예산, 기간, 작품 등 명시)
2. 예시 참고: "인상주의 특별전을 3000만원 예산으로 기획해줘"
3. 다시 실행

---

### 문제 5: Canvas 노드가 보이지 않습니다
**원인**: Canvas 동기화 실패 또는 Phase 미완료

**해결**:
1. 우측 패널의 "Canvas" 탭 클릭
2. 페이지 새로고침
3. 해당 워크플로우 다시 실행
