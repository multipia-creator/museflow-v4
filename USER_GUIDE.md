## 🎨 **주요 기능**

### **1. AI 추천 🤖**
**위치**: 네비게이션 바 (보라+핑크 버튼)

**기능**:
- 긴급 프로젝트 자동 감지 (D-7 이하)
- 예산 초과 프로젝트 경고
- 장기 미진행 프로젝트 알림
- 다음 단계 준비 완료 프로젝트 추천

**사용법**:
1. 네비게이션 바에서 **"AI 추천"** 버튼 클릭
2. 추천 목록 확인
3. **"확인하기"** 버튼으로 프로젝트 열기

---

### **2. 즐겨찾기 ⭐**
**위치**: 프로젝트 카드 좌상단 + 네비게이션 바

**기능**:
- 중요한 프로젝트를 빠르게 접근
- 원클릭 필터링
- 자동 저장 (브라우저 재시작 후에도 유지)

**사용법**:
1. 프로젝트 카드 좌상단의 **별 아이콘** 클릭
   - ☆ 빈 별 → ⭐ 채워진 별
2. 네비게이션 바의 **"즐겨찾기"** 버튼으로 필터링

---

### **3. 예산 관리 💰**
**위치**: Budget 페이지 또는 Dashboard 편집 모달

**기능**:
- 예산 입력 (총 예산, 사용 예산)
- 실시간 사용률 표시
- 예산 초과 경고
- 예산 변경 이력 추적
- Excel/CSV 보고서 생성

**사용법 A (Dashboard에서)**:
1. 프로젝트 카드 → **"수정"** 버튼
2. **"💰 예산 정보"** 섹션 입력
3. **"저장"** 버튼 클릭

**사용법 B (예산 페이지에서)**:
1. Budget 페이지 접속
2. 테이블에서 **"수정"** 버튼
3. 값 입력 후 **"저장"**
4. **"이력"** 버튼으로 변경 내역 확인

---

### **4. Canvas V3 🎨**
**위치**: 프로젝트 카드 → "보기" 버튼

**기능**:
- 88개 박물관 전문 노드
- 드래그앤드롭 워크플로우 편집
- AI 자동 생성
- 자동 저장 (10초마다)

**사용법**:
1. 프로젝트 카드 → **"보기"** 버튼
2. 좌측 패널에서 **노드 선택**
3. 캔버스에 **드래그앤드롭**
4. 노드 연결 (출력 → 입력)
5. 자동 저장됨 (하단 상태 표시)

---

### **5. 워크플로우 도구 🛠️**
**위치**: Workflow Tools 페이지

**6가지 도구**:
1. **템플릿 선택**: 10개 사전 정의 템플릿
2. **고급 필터**: 날짜/키워드/큐레이터/위치
3. **저장된 검색**: 검색 조건 저장 및 로드
4. **Export/Import**: JSON, CSV, Excel
5. **Canvas V3**: 노드 에디터
6. **일괄 작업**: 여러 프로젝트 동시 편집

## 📋 **단계별 가이드**

### **시나리오 1: 새 전시 프로젝트 만들기**

**Step 1: 템플릿 선택**
1. Dashboard → **"새 프로젝트"**
2. **"전시 관리"** 템플릿 선택

**Step 2: 기본 정보 입력**
- 프로젝트 제목: "한국 도자기 특별전"
- 유형: 전시
- 단계: 기획
- 큐레이터: 홍길동
- 장소: 1전시실

**Step 3: 예산 입력**
- 총 예산: ₩50,000,000
- 사용 예산: ₩0 (시작 단계)
- 시작일: 2025-01-01
- 종료일: 2025-03-31

**Step 4: 워크플로우 확인**
- Canvas V3가 자동으로 열림
- AI가 전시 워크플로우를 자동 생성
- 노드 추가/수정 가능

**Step 5: 즐겨찾기 추가**
- 프로젝트 카드 좌상단 ⭐ 클릭

---

### **시나리오 2: 긴급 프로젝트 확인하기**

**Step 1: AI 추천 확인**
1. 네비게이션 바 → **"AI 추천"** 버튼
2. 긴급 프로젝트 목록 확인

**Step 2: 프로젝트 열기**
- **"확인하기"** 버튼 → Canvas V3 자동 열림

**Step 3: 마감일 확인**
- 프로젝트 상세에서 D-day 확인

**Step 4: 액션 취하기**
- 워크플로우 진행 상황 업데이트
- 팀원에게 알림 전송

---

### **시나리오 3: 예산 보고서 만들기**

**Step 1: 예산 페이지 접속**
- Navigation → **"Budget"**

**Step 2: Excel 보고서 생성**
1. 우측 상단 **"Excel 보고서"** 버튼
2. 4개 시트 자동 생성:
   - 요약
   - 프로젝트 목록
   - 유형별 통계
   - 월별 추이
3. 파일 다운로드: `예산관리_보고서_2025-12-01.xlsx`

**Step 3: 차트 이미지 다운로드**
- 각 차트의 **"내보내기"** 버튼
- PNG 이미지로 저장

## ❓ **자주 묻는 질문**

### **Q1: 온보딩 투어를 다시 보고 싶어요**
```javascript
// 브라우저 콘솔에서 실행
onboardingTour.reset();
location.reload();
```

### **Q2: 즐겨찾기가 사라졌어요**
- 즐겨찾기는 브라우저 LocalStorage에 저장됩니다
- 브라우저 데이터를 삭제하면 사라집니다
- 다시 별을 클릭하여 추가하세요

### **Q3: AI 추천이 없어요**
- AI 추천은 다음 조건에서 나타납니다:
  - D-7 이하 프로젝트 있음
  - 예산 초과 프로젝트 있음
  - 30일+ 업데이트 없는 프로젝트 있음
  - 준비 완료 프로젝트 있음

### **Q4: 예산 이력을 보고 싶어요**
1. Budget 페이지 → 테이블
2. 해당 프로젝트의 **"이력"** 버튼 클릭
3. 타임라인 모달에서 모든 변경 내역 확인

### **Q5: 프로젝트를 백업하고 싶어요**
1. Workflow Tools → **"Export/Import"**
2. **"JSON"** 버튼 클릭
3. 전체 시스템 백업 다운로드

## 💡 **팁 & 트릭**

### **생산성 향상**

**Tip 1: 즐겨찾기 활용**
- 중요한 프로젝트를 즐겨찾기에 추가

- 유사한 프로젝트는 템플릿으로 저장
- 프로젝트 카드 → "템플릿" 버튼

**Tip 4: 검색 조건 저장**
- 자주 사용하는 필터 조건을 저장
- Workflow Tools → "저장된 검색"

**Tip 5: 예산 이력 활용**
- 매월 예산 변경을 이력으로 추적
- 감사 자료로 활용 가능

### **고급 기능**

**Canvas V3 단축키**
- `Space + Drag`: 캔버스 이동
- `Ctrl + Wheel`: 확대/축소
- `Delete`: 선택한 노드 삭제
- `Ctrl + S`: 수동 저장

**일괄 작업**
1. Dashboard에서 여러 프로젝트 선택
2. Workflow Tools → "일괄 작업"
3. 단계 일괄 변경, 유형 변경, 삭제 등

**고급 필터**
- 날짜 범위: 정확한 기간 지정
- 키워드: 제목, 설명, 큐레이터, 위치 모두 검색
- 빠른 필터: 이번 주, 이번 달, 긴급

## 🔔 **V10.0 신규 기능** 🆕

### **1. 알림 자동화 시스템**
**자동으로 알려드립니다**:
- ⚠️ 예산 80% 사용 시 경고
- 🚨 예산 100% 초과 시 긴급 알림
- 📅 D-7 마감일 알림
- ⏰ D-1 최종 알림
- ⏸️ 30일+ 미진행 프로젝트 알림

**사용법**:
```javascript
// 콘솔에서 수동 체크
notificationAutomation.checkAllProjects()

// 알림 이력 확인
notificationAutomation.getHistory()
```

### **2. PWA (모바일 앱 설치)** 📱
**홈 화면에 추가하세요**:
1. Dashboard 접속
2. 우측 하단 **"앱 설치"** 버튼 클릭
3. 홈 화면에 추가
4. 오프라인에서도 작동!

### **3. 고급 AI 기능** 🤖
```javascript
// 유사 프로젝트 찾기
advancedFeatures.findSimilarProjects(1)

// 프로젝트 검색
advancedFeatures.searchProjects("도자기")

// 예산 최적화 제안
advancedFeatures.suggestBudgetOptimization()

// 워크로드 분석
advancedFeatures.analyzeWorkload()

// 분석 리포트 내보내기
advancedFeatures.exportAnalyticsReport()
```

### **4. 협업 기능** 👥
```javascript
// 공유 링크 생성
advancedFeatures.generateShareLink(1)

// 댓글 추가
advancedFeatures.addComment(1, "검토 완료!")

// 댓글 확인
advancedFeatures.getComments(1)
```
