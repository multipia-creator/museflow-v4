## 📝 사용 가이드

### **1. Dashboard 사용법**

```
1. 로그인 → Dashboard 자동 이동
2. "오늘의 미션" 섹션 확인
   - D-7 이내 긴급 전시 표시
   - [미션 보기] 클릭 → Canvas 이동
3. "이번 주 타임라인" 확인
   - 주간 일정 한눈에 파악
4. "진행 중인 전시들" 카드 클릭
   - 원하는 전시 선택 → Canvas 이동
5. "팀 활동" 피드 확인
   - 팀원들의 최근 작업 확인
```

### **2. Canvas 사용법**

```
1. Dashboard에서 전시 카드 클릭
2. 3가지 뷰 중 선택:
   
   [타임라인]
   - 5단계 프로젝트 흐름 확인
   - 단계별 작업 파악
   
   [칸반보드]
   - 작업 상태별 정리
   - Drag & Drop으로 이동 (향후 지원)
   
   [갤러리]
   - 전시 상세 정보 확인
   - 메타데이터 한눈에 보기

3. [← 대시보드] 클릭 → Dashboard 복귀
```
