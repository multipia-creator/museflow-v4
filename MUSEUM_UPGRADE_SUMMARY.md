```sql
-- 뮤지엄 메타데이터

budget_used INTEGER          -- 사용 예산
artwork_count INTEGER        -- 작품 수량

-- 시각화
thumbnail_url TEXT           -- 썸네일 이미지
color_tag TEXT               -- 색상 태그
```

**인덱스:**
- `idx_projects_dates` (start_date, end_date)
- `idx_projects_type` (type)
- `idx_projects_phase` (phase)

**로컬 적용:** ✅ 완료
